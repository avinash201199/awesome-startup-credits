## Stacking Strategy

Smart founders **sequence** applications to maximize runway. A common path:

1. **Phase 1 — Bootstrapped (no VC):**
   - NVIDIA Inception (free, unlocks better terms elsewhere)
   - Microsoft Founders Hub ($1K–$5K Azure, self-serve)
   - AWS Activate self-serve tier ($1K)
   - Stripe Atlas / Mercury / Brex (cascade-unlock partner deals)
   - **~$25K–$40K claimed with no funding**

2. **Phase 2 — Post-funding (within 12 months of round):**
   - AWS Activate Portfolio ($100K) —  *hard 12-month deadline*
   - Google Cloud Scale ($200K+)
   - Microsoft Investor Network ($150K)

3. **Phase 3 — AI & specialty:**
   - Anthropic Startup ($25K+ Claude), OpenAI Tier 2/3
   - Datadog, HubSpot, Vanta/Drata, MongoDB

> Apply to **multiple programs simultaneously**. Many require accelerator/VC affiliation only for higher tiers — get the easy ones first.

## Application Tips

- **Use a company-domain email** (not Gmail/Outlook) — the #1 rejection reason.
- **Have a real website** — thin or placeholder sites get rejected.
- **Keep Crunchbase/PitchBook consistent** — mismatched info triggers rejection.
- **Read the billing fine print** — some programs (AWS) charge for usage above credits; others just cut you off.
- **Track expiration dates** — most credits expire in 12–24 months. Keep a spreadsheet of approvals, amounts, and renewal terms.
- **Don't build critical dependencies on expiring credits** — use them for dev, testing, and experimentation.
- **Beware stacking caveats** — e.g., claiming AWS via Brex/Mercury may reduce your direct AWS Activate amount.
